# **App Name**: ProjectVerse

## Core Features:

- Admin Authentication: Secure username/password login for admin accounts.
- Faculty Authentication: Secure username/password login for faculty accounts.
- Batch Authentication: Secure username/password login for batch accounts.
- Quota Management: Manage project coordinator quotas assigned to student batches.
- Problem Statement Upload: Admins and faculty members can upload Problem Statements (PS).
- Problem Statement Display: Display PSs for batch students with selection restrictions.
- Problem Statement Details: View PS details and project coordinator information post-selection.

## Style Guidelines:

- Primary color: Deep purple (#ad005c), evoking feelings of intellect and creativity.
- Background color: Light lavender (#f9fafb), a soft, desaturated purple to create a calm atmosphere.
- Accent color: Soft blue (#001c70), which is analogous to purple and offers a refreshing contrast.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, providing a modern yet readable interface.
- Use clean, outline-style icons to represent different functionalities, ensuring clarity and ease of use.
- Maintain a consistent grid-based layout across all screens, optimizing content presentation and responsiveness.
- Employ subtle transitions and animations to enhance user interaction and provide feedback on actions.
